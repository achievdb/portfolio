Project 1: Linux server

Beschrijving

Voor dit project heb ik een Ubuntu Server geïnstalleerd in een virtuele machine. Vervolgens heb ik gewerkt met gebruikersbeheer, groepen en toegangsrechten binnen een Linux-omgeving.

Door verschillende configuraties uit te testen, kreeg ik inzicht in hoe rechten en eigenaarschap invloed hebben op systeembeveiliging. Tijdens dit proces liep ik tegen enkele problemen aan, die ik zelfstandig heb opgelost. Dit heeft mijn praktische kennis van Linux aanzienlijk versterkt.

Wat ik gedaan heb

  • Ubuntu Server installeren
  • Gebruikers aanmaken
  • Apache installeren
  • Server testen
